
# Food Delivery SQL Mini Project

## Overview
This mini SQL project simulates a simple food delivery service database (similar to Zomato/Swiggy).  
It includes database schema, sample data, and analytical SQL queries.  
You can upload this project directly to GitHub.

## Files Included
- **schema_and_queries.sql** – Contains table creation, inserts, and SQL queries.
- **README.md** – Documentation for the project.

## Features
✔ Database schema  
✔ Clean sample data  
✔ Analytical queries (basic → advanced)  
✔ Real-world data relationships  
✔ Ready for GitHub upload  

## How to Use
1. Import the SQL file into MySQL,
2. Run the queries to analyze ordering behavior, revenue, delivery efficiency, etc.
3. Extend the schema or add new queries as needed.
